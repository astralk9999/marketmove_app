## A. INTRODUCCIÓN

### ¿Qué vamos a desarrollar?

Estimado cliente de MarketMove S.L.,

Nos complace presentarle este presupuesto para el desarrollo de su aplicación móvil de gestión comercial. Hemos analizado sus necesidades y le proponemos una solución moderna, intuitiva y fácil de usar.

**Su aplicación permitirá:**
- Apuntar las ventas del día de forma rápida y sencilla
- Registrar todos los gastos organizados por categorías
- Llevar un control del inventario con alertas cuando el stock esté bajo
- Ver en tiempo real cuánto está ganando (ventas menos gastos)
- Gestionar los productos de su tienda con precios y cantidades

### ¿Por qué Flutter?

Hemos elegido **Flutter** como tecnología de desarrollo por las siguientes razones:

| Ventaja | Beneficio para usted |
|---------|---------------------|
| **Una sola app para todo** | Su aplicación funcionará tanto en Android como en iPhone sin coste adicional |
| **Rapidez** | La app será muy fluida y responderá al instante |
| **Diseño moderno** | Interfaces atractivas que sus empleados disfrutarán usar |
| **Mantenimiento sencillo** | Futuras actualizaciones serán más económicas |
| **Tecnología de Google** | Respaldo de una de las mayores empresas tecnológicas |

### Base de datos: Supabase

Para almacenar sus datos de forma segura usaremos **Supabase**, que ofrece:
- Almacenamiento seguro en la nube
- Acceso desde cualquier dispositivo
- Copias de seguridad automáticas
- Sistema de usuarios con contraseñas cifradas

## B. FASES DEL PROYECTO

### Fase 1: Análisis y Toma de Requisitos
- Reunión inicial con el cliente
- Definición de funcionalidades
- Documentación de requisitos
- **Duración estimada:** 4 horas

### Fase 2: Diseño UX/UI Básico
- Bocetos de las pantallas principales
- Diseño de la navegación
- Selección de colores y tipografías
- Prototipo visual
- **Duración estimada:** 8 horas

### Fase 3: Arquitectura del Proyecto
- Estructura de carpetas profesional
- Configuración de proveedores de estado
- Diseño del modelo de datos
- Configuración inicial del proyecto
- **Duración estimada:** 6 horas

### Fase 4: Desarrollo del Frontend en Flutter
- Pantalla de precios y registro
- Sistema de autenticación (login/registro)
- Pantalla de ventas
- Pantalla de gastos
- Pantalla de productos y stock
- Panel de resumen (balance)
- Panel de administración
- **Duración estimada:** 40 horas

### Fase 5: Integración con Base de Datos (Supabase)
- Configuración de Supabase
- Creación de tablas
- Políticas de seguridad (RLS)
- Conexión app-base de datos
- Sistema de roles de usuario
- **Duración estimada:** 16 horas

### Fase 6: Pruebas Funcionales
- Pruebas de cada funcionalidad
- Corrección de errores
- Pruebas de rendimiento
- Validación con el cliente
- **Duración estimada:** 12 horas

### Fase 7: Documentación Final
- Manual de usuario
- Documentación técnica
- Guía de instalación
- **Duración estimada:** 6 horas

### Fase 8: Entrega y Publicación
- Preparación para tiendas (Play Store/App Store)
- Configuración de certificados
- Publicación (mock/real según acuerdo)
- Formación básica al cliente
- **Duración estimada:** 8 horas
